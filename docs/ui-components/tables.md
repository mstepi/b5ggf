## 公序良俗に反する契約
反社会的な契約などは守る必要がない
<br>そのような契約は<span style="color:rgb(248, 105, 105);">最初から無効</span>

## 第三者への抵抗
公序良俗に反する契約内容を守らせるわけには行かないため、
<br>契約の無効は善意の<span style="color:rgb(248, 105, 105);">第三者にも抵抗ができる</span>
